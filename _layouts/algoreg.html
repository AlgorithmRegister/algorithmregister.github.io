Om Algoritmeregister in de praktijk te toetsen en verder te ontwikkelen zoeken we testorganisaties. Ben je bezig met het inventariseren van algoritmische toepassingen in jouw organisatie en wil je Algoritmeregister daarvoor inzetten? We komen graag in contact!

Achtergrond

Dat overheden techniek niet altijd verantwoord inzetten is regelmatig in het nieuws terug te vinden.

Onder de vlag Publieke Controle op Algoritmes werkt een breed consortium van verschillende overheden aan instrumenten om de controle op de inzet van algoritmische toepassingen te vergroten.

In opdracht van het consortium wordt Algoritmeregister ontwikkeld: een publiek register waarin de algoritmische toepassingen die door overheden worden ontwikkeld en ingezet op een gestandaardiseerde, begrijpelijke manier worden beschreven en toegelicht.

Metadata Standaard

Het register en de standaard worden in wisselwerking met elkaar ontwikkeld. De huidige versie van de standaard is versie 0.2 (ook beschikbaar als v0.2 json schema en v0.2 csv template). Deze maturity van de metadata standaard is prototype.

Open (Source, Design, Development, Community)

Het register en de standaard worden open ontwikkeld volgens de architectuur principes van Common Ground en de Standard for Public Code. Alle code is te vinden op Github.

Let op: de maturity van de code base is prototype en de code bases zijn nog niet vrijgegeven op basis van open source licentie.

De ontwikkeling van de standaard en het register willen we in open samenwerking doen. We werken daarom in the open in de community van Code for NL. Praat mee op chat.codefor.nl in kanaal "Algoritmeregister", of volg de ontwikkelingen via Twitter.
